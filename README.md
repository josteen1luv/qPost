# qPost
FullStack app where you can sign up and log into your account to write, read, delete, edit and like posts.

## Technology Stack:

---

### Frontend:
+ React
+ Redux ToolKit / RTK Query
+ ReactRouterDom
+ Material UI
+ Formik and Yup
+ Axios
+ SCSS (SASS)
### Backend:
+ NodeJS
+ ExpressJS
+ Json Web Token
+ Bcrypt
+ Nodemailer
+ PostgreSQL
+ Sequelize
